# Standard Tier

The Clary AI Standard tier includes pre-integrated Phi-4 Multimodal model and provides enhanced document processing capabilities.

## Features

- **Document Processing**: Advanced document parsing and extraction
- **Pre-integrated AI Model**: Phi-4 Multimodal model for enhanced analysis
- **Workflow Engine**: Create complex document processing workflows
- **API Access**: Standard API access for integration
- **Self-Hosted**: Deploy on your own infrastructure
- **Email Support**: Access to email support

## Use Cases

- **Medium-sized Teams**: Ideal for medium-sized teams with advanced document processing needs
- **Business Applications**: Great for business applications that require AI-powered document analysis
- **Integration Projects**: Perfect for projects that need to integrate document processing into existing systems
