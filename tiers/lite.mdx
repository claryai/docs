# Lite Tier

The Clary AI Lite tier is a free, self-hosted solution that provides basic document processing capabilities without pre-integrated AI models.

## Features

- **Document Processing**: Basic document parsing and extraction
- **Workflow Engine**: Create simple document processing workflows
- **API Access**: Limited API access for integration
- **Self-Hosted**: Deploy on your own infrastructure
- **Community Support**: Access to community forums and documentation

## Use Cases

- **Small Teams**: Ideal for small teams with basic document processing needs
- **Developers**: Great for developers who want to experiment with document processing
- **Personal Projects**: Perfect for personal projects and learning
