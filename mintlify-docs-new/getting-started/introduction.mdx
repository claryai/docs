# Welcome to Clary AI

Clary AI is a powerful document processing platform that leverages open-source AI models with agentic capabilities to transform how you work with documents.

## What is Clary AI?

Clary AI is a self-hosted document processing platform that offers:

- **Document Processing**: Extract text, tables, and structured data from various document formats
- **AI-Powered Analysis**: Leverage AI models to understand document content
- **Workflow Automation**: Create custom workflows for document processing
- **API Access**: Integrate document processing capabilities into your applications

## Tiered Approach

## Key Features

### Document Processing

Clary AI can process a wide range of document formats, including:

- PDF documents (including scanned PDFs with OCR)
- Microsoft Office documents (Word, Excel, PowerPoint)
- Images (JPG, PNG, TIFF)
- HTML and web pages
- Plain text files

### AI-Powered Analysis

Depending on your tier, Clary AI can:

- Extract structured data from unstructured documents
- Classify documents by type and content
- Identify entities such as people, organizations, and locations
- Answer questions about document content
- Summarize long documents

### Workflow Automation

Create custom document processing workflows that can:

- Automatically route documents based on content
- Extract specific information from documents
- Generate reports and notifications
- Integrate with your existing systems

### API Access

Integrate Clary AI into your applications with our comprehensive API:

- RESTful API for easy integration
- Webhook support for event-driven architectures
- SDKs for popular programming languages
- Comprehensive documentation and examples
