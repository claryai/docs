## What is Clary AI?

Clary AI is a self-hosted document processing platform that leverages open-source AI models with agentic capabilities. It allows you to:

- Extract text, tables, and structured data from various document formats
- Leverage AI models to understand document content
- Create custom workflows for document processing
- Integrate document processing capabilities into your applications

## Available Tiers

Clary AI is available in three tiers:

- **Lite**: Basic document processing without pre-integrated LLM
- **Standard**: Enhanced capabilities with Phi-4 Multimodal integration
- **Professional**: Advanced features with cloud LLM connections
