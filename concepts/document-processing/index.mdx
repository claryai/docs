# Document Processing

Clary AI provides powerful document processing capabilities to extract and analyze information from various document types.

## Supported Document Types

- PDF documents
- Word documents (.docx, .doc)
- Excel spreadsheets (.xlsx, .xls)
- Images (.png, .jpg, .jpeg)
- Text files (.txt)
- HTML files

## Processing Pipeline

1. **Document Upload**: Upload documents via the web interface or API
2. **Preprocessing**: Clean and normalize document content
3. **Text Extraction**: Extract text content from documents
4. **Structure Analysis**: Identify document structure (headings, paragraphs, tables)
5. **Entity Extraction**: Identify and extract key entities and information
6. **Post-processing**: Format and structure the extracted information

## Features

### Text Extraction

Clary AI uses advanced OCR and text extraction techniques to accurately extract text from documents, including scanned PDFs and images.

### Table Extraction

Automatically identify and extract tables from documents, preserving their structure and relationships.

### Form Field Detection

Detect and extract form fields and their values from structured documents.

### Layout Analysis

Analyze document layout to understand the relationships between different elements.
