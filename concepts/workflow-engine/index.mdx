# Workflow Engine

The Clary AI Workflow Engine allows you to create custom document processing workflows.

## Key Concepts

- **Workflows**: A sequence of processing steps applied to documents
- **Tasks**: Individual processing steps within a workflow
- **Checkpoints**: Save points within a workflow for resumability
- **Templates**: Reusable workflow definitions

## Creating Workflows

Workflows can be created through the web interface or via the API.
