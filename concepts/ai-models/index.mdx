# AI Models

Clary AI leverages open-source AI models for document processing and analysis.

## Supported Models

- **Text Extraction**: Models for OCR and text extraction
- **Entity Recognition**: Models for identifying entities in documents
- **Classification**: Models for document classification
- **Summarization**: Models for document summarization

## Model Integration

Clary AI provides a flexible model integration framework that allows you to use different models for different tasks.
