# Authentication

All API requests to Clary AI require authentication using an API key.

## Obtaining an API Key

API keys can be obtained from the Clary AI dashboard.

1. Log in to your Clary AI account
2. Navigate to Settings > API Keys
3. Click "Generate New API Key"
4. Copy and securely store your API key

## Using Your API Key

Include your API key in the `Authorization` header of all API requests:

```bash
curl -X GET https://api.claryai.com/v1/documents \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## API Key Security

- Keep your API key secure
- Do not share your API key in public repositories
- Rotate your API key regularly
- Use environment variables to store your API key in applications
