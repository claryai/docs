# Documents API

The Documents API allows you to upload, retrieve, and manage documents in Clary AI.

## Upload a Document

```bash
curl -X POST https://api.claryai.com/v1/documents \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -F "file=@document.pdf" \
  -F "metadata={\"name\":\"Example Document\",\"tags\":[\"invoice\",\"finance\"]}"
```

## List Documents

```bash
curl -X GET https://api.claryai.com/v1/documents \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Get Document Details

```bash
curl -X GET https://api.claryai.com/v1/documents/{document_id} \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Delete a Document

```bash
curl -X DELETE https://api.claryai.com/v1/documents/{document_id} \
  -H "Authorization: Bearer YOUR_API_KEY"
```
