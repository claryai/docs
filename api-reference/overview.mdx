# API Overview

The Clary AI API allows you to integrate document processing capabilities into your applications.

## Authentication

All API requests require authentication using an API key.

```bash
curl -X GET https://api.claryai.com/v1/documents \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Base URL

The base URL for all API requests is:

```
https://api.claryai.com/v1
```

## Rate Limits

- **Lite Tier**: 100 requests per minute
- **Standard Tier**: 500 requests per minute
- **Professional Tier**: 1000 requests per minute
